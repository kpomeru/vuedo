# Vuedo by KpOmeru

This template should help get you started developing with Vue 3 in Vite.

## Project Setup, Build and Production

```sh
yarn install
```

### Development

```sh
yarn dev (development)
```

### Production

```sh
yarn build (production)
```

## Contribution
